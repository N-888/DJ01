# Dj01
Приложение Django, в котором при переходе на страницу data и test открываются страницы с разным содержимым, а на каждой из этих страниц дополнительно есть ссылки для перехода между страницами (на соседнюю страницу — одну из этих двух):
1. -> http://127.0.0.1:8000/data/ ←
2. -> http://127.0.0.1:8000/test/ ← 
3. -> Переход по ссылкам между страницами ←
